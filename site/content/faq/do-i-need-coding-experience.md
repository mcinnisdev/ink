You'll need basic familiarity with the command line and Markdown syntax, but you don't need to be an experienced developer. Ink's CLI handles the scaffolding, and most day-to-day work involves editing Markdown files and adjusting CSS variables. You don't need to understand Eleventy's internals or write any JavaScript to build and maintain a site. Check out our [getting started guide](/docs/getting-started/) to see how quick the setup process is.
