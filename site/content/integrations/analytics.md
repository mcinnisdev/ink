## Replacements

| WP Plugin | Ink Replacement | How to Embed | Free? | Ease |
|---|---|---|---|---|
| MonsterInsights | [Plausible Analytics](https://plausible.io) | One script tag in base template | $9/mo | Easy |
| Google Analytics WP | [Google Analytics (direct)](https://analytics.google.com) | GA4 script tag -- no plugin needed | Yes | Easy |
| Jetpack Stats | [Fathom Analytics](https://usefathom.com) | One script tag in base template | Paid | Easy |
| Hotjar WP Plugin | [Hotjar (direct)](https://hotjar.com) | Paste tracking script in base template | Limited | Easy |

## Recommended: Plausible

Privacy-friendly, lightweight, GDPR compliant with no cookie banners required. Open-source if you want to self-host. Add a single script tag to your Ink base layout and you're done.

Static Ink sites are naturally faster than WordPress -- your Core Web Vitals improve just from the migration, which directly benefits SEO rankings.
